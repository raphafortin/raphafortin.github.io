# Pixelz Games Website

Modern game studio website for [Pixelz Games](https://pixelzgames.com), showcasing their catalog: **CO-OP: Decrypted**, **Cefore**, and **Barista: Rise and Grind**.

## Games

| Game | Status | Year | Steam |
|------|--------|------|-------|

| Cefore | Shipped | 2018 | [Store Page](https://store.steampowered.com/app/625730/Cefore/) |
| CO-OP: Decrypted | Shipped | 2015 | [Store Page](https://store.steampowered.com/app/315130/COOP__Decrypted/) |

## Tech Stack

- **HTML5** – Semantic structure
- **CSS3** – Custom properties, grid, animations
- **Vanilla JS** – Scroll reveal, mobile menu, smooth scroll

All game images and data are sourced from Steam's CDN.

## Development

1. Open `index.html` in a browser, or
2. Run a local server:
   ```bash
   npx serve .
   ```

## Deployment

Static site. Deploy to any host (Netlify, Vercel, GitHub Pages, etc.) by uploading the project folder.
